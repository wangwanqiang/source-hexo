# 安装与配置

## windows

### 两个前提和条件
1. 首先要安装 Node.js 
2. 当然了还要安装 git

### 然后在Windows命令行输入

1. 安装hexo  
~~~
npm install -g hexo
~~~
2. 初始化一个blog，这里名字就叫 hexo
~~~
hexo init hexo
~~~
3. 进入hexo目录
~~~
 cd hexo
~~~
4. 安装依赖文件
~~~
npm install
~~~

5. 部署形成文件
~~~
hexo generate
~~~
